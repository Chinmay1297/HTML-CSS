Observations:

  • Margin is applied horizontally
  • Padding to border is not pushing away the words beside it.
  • Block elements push content away when margin/padding is used.
  • h1 is a block element
  • Inline elements doesnt push content away.
  • span is an inline element
  • Width and Height are not respected for inline elements, but respected for block elements.

What is inline-block then ?

  • Its hybrid of inline and block.
  • A block that doesnt break into a new line.
  • Padding and Margin push other elements away from the box

What is display:none ?

It completely removes the element from html.

Opacity just hides it, it will still consume space on the UI.
